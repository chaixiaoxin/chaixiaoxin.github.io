SEOer/SEMer选择长尾关键词的6个方向和拓词手法

发表于 2020-06-10 | 更新于: 2020-06-10 |
无论是做seo优化还是sem付费推广目的都是为了流量，而流量的是一切生意的本质。前排展现的工作就最为重要，为了得到更多的展现，长尾关键词的选择是推广工作的前期规划，长尾关键词的特征普遍存在网站的内容的标题、内容的页面和内容中。长尾关键词的搜索量前期可能不大，但这部分的流量不可忽视，做好长尾关键词的访客对应承载页面内容方案，长期会带来网站权重的提升。
总结了6种主流的长尾词选词方向：
  1. 与网站内容相关长尾关键词
  2. 选择用户可能来查询的长尾关键字
  3. 能解决搜索用户痛点的关键词
  4. 分析竞争对手的网站的长尾关键词
  5. 通过搜索引擎搜索后的相关搜索选择长尾关键词
  6. 利用长尾关键词挖掘工具确定长尾词。
拓展长尾关键词词库

​ 拓词围绕网站业务、关键字为核心，前后增加相关字

  1. 借助拓词工具（百度推广助手、5118和爱站等）进行关键词挖掘
  2. 查看搜索指数关联的关键字
  3. 百度搜索框上面关联的搜索词
  4. 结合用户搜索习惯，在核心业务产品词添加问句，比如：XXX好不好？那里可以买到XXX？如何选择XXX？等
  5. 区域拓词，不同省份城市的人他的搜索，如广东XXX代理、四川XXX代理
  6. 季节拓词法，如夏季XXX效果怎么样？
  7. 结合热点热词的长尾词。
长尾关键词优化
  1. 利用excel进行词库分类、排序，方便做数据分析，如分析搜索词意向在第几阶段，作为竞价出价的重要参数之一。
  2. 结合文章解决访客的内容出现的长尾关键词可做特别处理，如加粗，强化感受提升满意度。
以上，文章讲述从长尾关键词选择的方向，到怎么有效拓词，最后怎么进行优化网站的过程和方法。

记录我从0搭建Blog

发表于 2020-06-09 | 更新于: 2020-06-10 |

前言

自从接触互联网就幻想有一个属于自己的网站，记录着生活小事和学习过程的总结。而此次成功建站还是多得互联网大神劳作的成果和经验分享。本BLOG基于Node.js利用Hexo博客框架生成部署在Github上，[Hexo][1]是快速、简洁且高效的博客框架，自身有很多插件的支持，包括有数百个主题可选，更换方式非常简单。下面记录一下创建的过程和怎么部署到github上。

1. 前期准备

安装node.js、github及账号注册
1. node.js  [链接](https://nodejs.org/zh-cn/download/ "链接")
2. github    [github](http://github.io "github")下载并注册账号,和设置好公钥
3. 打开Gitbash
4. 输入：＄node -v #查询node.js版本号
5. 输入：＄npm -v  #查询npm管理工具版本号
6. 两者都可查询到版本号，证明环境部署完成。
创建的过程
1.由于hexo基于node.js
2.＄npm install -g hexo
3.＄cd D: #选择在D盘位置，看个人习惯
4.＄mkdir Myblog  #创建一个文件存放博客文件
5.＄hexo init  #初始化博客在当前Myblog文件下，也可在上一步直接＄hexo init myblog
6.＄hexo g  #生成网站
7.＄hexo s  #启动服务器，启动后可在浏览器查看输入：http://localhost:4000/
部署
安装一键部署工具，deployer，git的设置修改_config.yml
找到deploy：
        type： git
        repository: git@github.com:itmifen/blog.git
        branch: master
＄npm install hexo-deployer-git --save
＄hexo deploy
最终效果
参考：http://chaixiaoxin.github.io
